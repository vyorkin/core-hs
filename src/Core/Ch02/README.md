# Notes and exercises

### Multiplication machine

```
n m 0 0
.......
n 0 0 t
```

1.

```
    n m d   t
==> n m d-1 t+1
    where d > 0
```

2.

```
    n m   0 t
==> n m-1 n t
    where m > 0
```
